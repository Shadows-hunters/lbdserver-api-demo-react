# Shadow hunters
The project contains two server:
- the main app: 3000
- the api server: 8080
#
## use:
```bash
cd lbdserver-api-demo-react
```
start main app: `npm start`

start api server: `node server.js`

start **ALL**:
```
npm run all
```
## deployment
1. `npm run build`
2. move build folder to server
3. inside build folder on server: `serve`
